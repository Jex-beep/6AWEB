What are Directives?

Directives are special instructions in frameworks like Angular that tell the application how to manipulate the DOM or change the behavior of elements.

Examples of Directives

  • ngIf: Conditionally shows or hides elements
  • ngFor: Repeats elements for each item in a list
  • ngClass: Dynamically applies CSS classes
